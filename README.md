# Fuel Consumption Tracker

Projeto inicial para controlar consumo de combustível, criado a partir de um design de Figma.

## Estrutura

- `index.html` - layout principal
- `styles.css` - estilos visuais
- `script.js` - lógica para registrar abastecimentos e atualizar o resumo

## Como usar

1. Abra a pasta `FuelConsumptionTracker` no VS Code.
2. Abra `index.html` no navegador ou use uma extensão de preview.
3. Registre abastecimentos para ver o resumo e histórico.
