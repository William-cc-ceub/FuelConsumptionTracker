Perguntas Frequentes

Meus dados são enviados para um servidor? ▼

Não! Todos os dados são armazenados localmente no seu navegador usando localStorage. Sua privacidade está garantida.

Posso usar em múltiplos dispositivos? ▼

Os dados ficam em cada dispositivo. Para sincronizar entre dispositivos, você pode fazer backup manual exportando/importando os dados.

Como faço backup dos meus dados? ▼

Abra o DevTools (F12), vá na aba Console e execute: localStorage. Você pode fazer screenshot ou exportar para um arquivo.

Preciso instalar algo? ▼

Não! Basta abrir no navegador. Para offline, baixe o arquivo HTML único e abra localmente.

Quais navegadores são suportados? ▼

Chrome, Firefox, Safari, Edge e qualquer navegador moderno que suporte ES6 e localStorage.

Posso contribuir com o projeto? ▼

Sim! O projeto é open-source. Visite nosso repositório no GitHub para contribuições.
