Perguntas Frequentes

Meus dados são enviados para um servidor? ▼

Não! Todos os dados são armazenados localmente no seu navegador usando localStorage. Nenhuma informação é enviada para servidores externos. Sua privacidade está completamente garantida.

Preciso criar uma conta? ▼

Não! Não há login, cadastro ou autenticação necessária. Basta abrir e começar a usar. É totalmente anônimo.

Posso usar em múltiplos dispositivos? ▼

Sim, você pode usar em vários dispositivos. Porém, os dados ficam isolados em cada um. Se você quer sincronizar entre dispositivos, você precisará fazer backup manualmente dos dados de um e restaurar em outro.

Como faço backup dos meus dados? ▼

Para fazer backup:

  1. Abra a aplicação
  2. Pressione F12 para abrir DevTools
  3. Vá na aba "Application" ou "Storage"
  4. Procure por "Local Storage"
  5. Você verá seus dados em formato JSON
  6. Copie e salve em um arquivo de texto seguro
E se limpar os cookies do navegador? ▼

⚠️ Aviso: Limpar cache/cookies/dados do navegador pode apagar seus dados. Faça backup antes de fazer limpeza de navegador!

Funciona offline? ▼

Sim! Especialmente se usar o arquivo app-standalone.html que é totalmente auto-contido. Uma vez carregado, funciona completamente offline.

Quais navegadores são suportados? ▼

Qualquer navegador moderno que suporte ES6 e localStorage:

  • ✅ Chrome/Chromium (versão 55+)
  • ✅ Firefox (versão 54+)
  • ✅ Safari (versão 11+)
  • ✅ Edge (versão 79+)
  • ✅ Opera (versão 42+)
Como instalo? ▼

Não precisa instalar! Você tem 3 opções:

  1. Online: Abra a aplicação web
  2. Offline: Baixe app-standalone.html e abra no navegador
  3. Servidor local: Execute o start-server.bat
Posso imprimir meu relatório? ▼

Sim! Você pode:

  • Usar Ctrl+P (ou Cmd+P no Mac) para imprimir a página
  • Screenshot dos gráficos e tabelas
  • Exportar os dados do localStorage para um arquivo
Posso editar ou deletar registros? ▼

Sim! Na tabela de registros, cada entrada tem botões de ação:

  • ✏️ Editar - modifica o registro
  • 🗑️ Deletar - remove o registro
Como calcula o consumo? ▼

Fórmula: (Total de litros ÷ Total de km) × 100 = L/100km

Exemplo: Se você gastou 50 litros em 500 km: (50 ÷ 500) × 100 = 10 L/100km

Quanto menor este número, mais eficiente é o consumo.

E se esquecer de registrar um abastecimento? ▼

Você pode adicionar registros retroativos:

  1. Clique em "Adicionar registro"
  2. Use uma data anterior
  3. Os dados serão agregados corretamente
Posso ter múltiplos veículos? ▼

Sim! Você pode cadastrar quantos veículos quiser em "Gerenciar veículos". Cada um terá:

  • Histórico independente
  • Seus próprios cálculos de consumo
  • Gráficos separados
Como contribuir com o projeto? ▼

O Fuel Consumption Tracker é open-source! Você pode:

  • Reportar bugs
  • Sugerir melhorias
  • Fazer pull requests no GitHub
  • Compartilhar com amigos
Tem uma versão para iOS/Android? ▼

A aplicação web funciona em smartphones! É responsiva e otimizada para celular. Você pode:

  • Abrir no navegador do celular
  • Adicionar à tela inicial (favorito)
  • Usar como um app nativo
