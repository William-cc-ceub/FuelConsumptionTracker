Política de Privacidade

Última atualização: 12 de junho de 2026

1. Introdução

Bem-vindo ao Fuel Consumption Tracker ("nós", "nosso" ou "a Aplicação"). Levamos sua privacidade muito a sério. Esta Política de Privacidade explica como coletamos, usamos, divulgamos e protegemos seus dados quando você usa nossa aplicação web.

🔒 Resumo Rápido: Todos os seus dados são armazenados localmente no seu navegador. Nós não enviamos nenhuma informação para servidores externos.

2. Que Dados Coletamos

Você escolhe que dados adicionar à aplicação. Potencialmente coletamos:

  • Informações sobre seus veículos (marca, modelo, placa, etc.)
  • Datas de abastecimento
  • Quantidade de combustível
  • Quilometragem percorrida
  • Preço do combustível
  • Observações pessoais

3. Como Armazenamos Seus Dados

Todos os dados são armazenados localmente no seu navegador usando a API localStorage do HTML5. Especificamente:

  • ✅ Os dados residem apenas no seu computador/dispositivo
  • ✅ Nenhum servidor externo acessa seus dados
  • ✅ Não fazemos backup automático em nuvem
  • ✅ Não há sincronização automática entre dispositivos

4. O Que Não Coletamos

A Aplicação NÃO:

  • Rastreia sua localização geográfica
  • Coleta dados de navegação
  • Usa cookies de terceiros
  • Envia dados para servidores analíticos
  • Coleta informações pessoais identificáveis (nome, email, CPF)
  • Usa publicidade ou rastreamento comportamental
  • Compartilha dados com terceiros

5. Segurança dos Dados

Sua segurança é importante. Como seus dados ficam localmente:

  • Isolamento: Dados isolados no seu navegador, inacessível a sites externos
  • Sem Transmissão: Dados nunca são enviados pela internet
  • HTTPS: Se usar online, sempre sobre conexão segura
  • Seu Controle: Você controla totalmente quando deletar dados
⚠️ Importante: Como os dados são locais, limpar cache/cookies do seu navegador pode resultar em perda de dados. Sempre faça backup!

6. Consentimento

Ao usar a Aplicação, você consente com esta Política de Privacidade. Se você não concorda com nossa política, por favor não use a Aplicação.

7. Direitos de Acesso e Portabilidade

Você tem direito de:

  • Acessar todos os seus dados a qualquer momento
  • Exportar seus dados em formato legível
  • Deletar seus dados quando quiser
  • Usar seus dados em outras aplicações

Como acessar seus dados:

  1. Abra a Aplicação
  2. Pressione F12 para abrir Developer Tools
  3. Vá na aba "Application" ou "Storage"
  4. Procure por "Local Storage"
  5. Você verá todos os seus dados em formato JSON

8. Retenção de Dados

Os dados são retidos enquanto você os manter no localStorage do seu navegador. Você pode deletar:

  • Registros individuais: Através da interface da Aplicação
  • Todos os dados: Clicando "Limpar tudo" na página principal
  • Cache do navegador: Através das configurações do seu navegador

9. Mudanças nesta Política

Podemos atualizar esta Política de Privacidade de tempos em tempos. A data de última atualização está no topo. Mudanças significativas serão comunicadas.

10. Contato

Se você tem questões sobre esta Política de Privacidade, por favor abra uma issue no GitHub ou entre em contato através de nossos canais de suporte.

11. Conformidade com Regulamentos

Esta Aplicação é projetada para estar em conformidade com:

  • 🇧🇷 LGPD (Lei Geral de Proteção de Dados - Brasil)
  • 🇪🇺 GDPR (Regulamento Geral de Proteção de Dados - Europa)
  • 🇺🇸 CCPA (California Consumer Privacy Act - EUA)

Como não coletamos dados pessoais identificáveis e não transmitimos informações, estamos naturalmente em conformidade com estes regulamentos.

12. Responsabilidade

A Aplicação é fornecida "como está", sem garantias. Você é responsável por:

  • Fazer backup regular dos seus dados
  • Manter a segurança do seu dispositivo
  • Manter seu navegador atualizado
